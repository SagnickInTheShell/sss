About Binomial Distribution

The Binomial Distribution models the number of successes in a fixed number of independent trials, each with the same probability of success.

Key Properties

  • n: Number of independent trials (must be fixed in advance)
  • p: Probability of success on each trial (constant across trials)
  • k: Number of successful outcomes we're calculating probability for
  • Mean (μ): Expected average number of successes = n × p
  • Variance (σ²): Measure of spread = n × p × (1 - p)

How Parameters Affect the Distribution

  • Increasing n makes the distribution approach a normal curve (Central Limit Theorem)
  • When p = 0.5, the distribution is symmetric
  • When p < 0.5, the distribution is right-skewed
  • When p > 0.5, the distribution is left-skewed
